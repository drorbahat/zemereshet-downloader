# 🎵 מוריד זמרשת

הורדת שירים מזמרשת **ישירות לתיקייה** - עם שמות עבריים מלאים!

## 📦 התקנה (פעם אחת)

### שלב 1: ודא ש-Node.js מותקן
```bash
node --version
```

אם לא מותקן, הורד מ: https://nodejs.org

### שלב 2: התקן את הספריות
```bash
cd zemereshet-downloader
npm install
```

זהו! ההתקנה הושלמה ✅

---

## 🚀 שימוש

### כל פעם שאתה רוצה להשתמש:

1. **הפעל את השרת:**
```bash
npm start
```

2. **פתח בדפדפן:**
`http://localhost:3000`

3. **הדבק:**
   - קישור לשיר: `https://www.zemereshet.co.il/song.asp?id=5197`
   - נתיב תיקייה: `/Users/שם_משתמש/Music/Zemereshet`

4. **לחץ "הורד לתיקייה"**

5. **זהו!** הקבצים נשמרו בתיקייה שבחרת עם שמות עבריים מושלמים! 🎉

## ⚙️ איך זה עובד?

השרת פועל על המחשב שלך ושומר קבצים **ישירות** למערכת הקבצים:
1. טוען את דף השיר מזמרשת
2. מחלץ את כל ההקלטות (MP3)
3. יוצר תיקייה חדשה עם שם השיר
4. שומר את כל הקבצים עם שמות עבריים מלאים
5. מוסיף קובץ מידע

**אין ZIP = אין בעיות encoding!** 💪
